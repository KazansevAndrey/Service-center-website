1. Проект состоит из более чем 3 логических разделов
2. Интерфейс каждого раздела должен содержит логотип и название системы, 
глобальное меню, навигационную информацию (текущий раздел), название раздела, 
несколько абзацев текста с картинками. 
3. На страницах проекта реализована «красивая таблица» средствами 
css и список с художественными маркерами (на страница ремонта техники). На сайте должны быть реализованы 
трансформации, переходы и анимации.
Трансформации и переходы: используются для увеличения иконок соц. сетей.
Анимации: анимирована 404 страница.
4. Проект включает в себя реализацию интерфейса фотогалереи. На главной странице присутствует блок "Сертификаты". 
При нажатии на сертификат открывается увеличенное изображение. 
Для закрытия увеличенного изображения необходимо нажать на большой сертификат.
5. В проекте реализованы динамическое содержимое с помощью скриптов на языке JavaScript. 
У каждой формы ввода есть проверка, если пользователь вводит неправильно значение, то появляется предупреждение. 
Реализован график chart.js на главной странице.
8. Проект опубликован на github.
